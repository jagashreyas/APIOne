# Playwright API Automation Framework (C#)

## Tech Stack
- C#
- Playwright
- NUnit

## Features
- GET & POST API testing
- Reusable API helper
- Model-based request body
- Clean project structure

## How to Run
1. Install dependencies:
   dotnet restore

2. Install Playwright:
   pwsh bin/Debug/net8.0/playwright.ps1 install

3. Run tests:
   dotnet test
